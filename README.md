## Getting Started

Follow the instructions based on the package manager you use (npm/pnpm)

### Install dependencies


```bash
pnpm install
```

```bash
brew install flac
```

### Run the newly created app

```bash
pnpm run dev
```

### That's it

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
